== Description ==

Allows the user to add a custom contact form.

== Integrations ==

This needs SMTP mailing settings to work.

Example: Postmark Wordpress Plugin

01. Signup: https://postmarkapp.com/
02. Setup sender signatures, adding DNS to domain
03. Setup server and grab API key
04. Request approval from Postmark
05. Download and install: https://wordpress.org/plugins/postmark-approved-wordpress-plugin/
06. Add API key, sender email to match sender signatures and enable
07. Send test email
